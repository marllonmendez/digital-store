## Sobre
Este é o frontend mockado de um e-commerce, desenvolvido como parte de um desafio proposto na formação Full-Stack do programa Geração Tech, em parceria com o governo do estado do Ceará.

## Guia de Instalação

<h4>1. Clonagem do Repositório</h4>

```bash
git clone https://github.com/marllonmendez/digital-store.git
```

<h4>2. Instalação de Dependências</h4>

```bash
npm install
```

<h4>4. Execução Local</h4>

```bash
npm run dev
```
